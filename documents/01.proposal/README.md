# 01.proposal/

> **Forward 모드(외부 발주·RFP 수주) 전용** 제안 진입자료(PD)를 둔다. 다른 모드에서는 비어 있거나 미생성된다.
> PD는 *진입자료(외부·오케 입력)*라 RACI 활동표에 생산 책임(R)이 없다 — 대신 입력 출처를 표시한다.

| 하위 | 무엇 | 출처 |
|---|---|:---:|
| `PD010_RFP/` | 발주처 제안요청서 | 발주처 |
| `PD015_기술협상서/` | 기술 협상 내용 | 발주처·오케 |
| `PD020_제안서/` | 수행사 제안서 | 수행사 |
| `PD030_계약/` | 계약서·과업지시서 | 발주처 |
| `PD040_변경요청/` | 계약 후 변경 요청 | 발주처 |

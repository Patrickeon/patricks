## 7. 제약 사항

| 제약 | 영향 | 처리 |
|------|------|------|
| Rust/Cargo 미탐지 | Tauri 앱 실제 구동 및 Rust 통합 테스트 실행 불가 | 정적 분석과 프론트 빌드 결과로 보완. 환경 구성 후 재실행 필요 |
| 실제 Redmine API 미호출 | 이슈 조회·생성·상태변경의 네트워크 동작 미검증 | IPC 체인과 컴파일 정합성만 확인 |
| 실제 AI Provider credential 없음 | bootTeam/HealthCheck의 provider 실제 인증 검증 미수행 | wrapper/route/store 연결만 확인 |
| 지정 산출물 경로가 기존 구조와 다름 | 기존 `04.테스트/TS-20_통합테스트결과서`가 아닌 `05.시험/TS-20_통합테스트결과서`에 작성 | PM 지시 경로 우선 적용 |

## 8. 결함 및 후속 조치

| ID | 내용 | 심각도 | 조치 |

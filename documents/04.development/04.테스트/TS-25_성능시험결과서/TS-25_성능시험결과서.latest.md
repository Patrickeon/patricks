## 6. IPC 호출 체인 복잡도

| 기능 영역 | 호출 시작점 | IPC wrapper | Rust command | 내부 처리 | 계층 | 판정 |
|-----------|-------------|-------------|--------------|-----------|:---:|:---:|
| 팀 부팅 | `BootView.vue` | `src/ipc/agent.ts` | `boot_team` | `AgentSessionService` | 4 | PASS |
| 워크스페이스 설정 저장 | `ProjectSettingsView.vue` | `src/ipc/workspace.ts` | `save_workspace_config` | file/config 처리 | 4 | PASS |
| 산출물 쓰기 | `DeliverableView.vue` | `src/ipc/document.ts` | `write_latest_document` | `FileDocumentRepository` | 4 | PASS |
| 레드마인 이슈 | `stores/redmine.ts` | `src/ipc/redmine.ts` | `redmine_*` | `redmine_client.rs` | 4 | PASS |
| API 키 저장 | `stores/settings.ts`, `stores/redmine.ts` | `src/ipc/credential.ts` | `save_credential` | OS credential store | 4 | PASS |
| HealthCheck | `SettingsView.vue` | `src/ipc/health.ts` | `run_health_check` | provider/tool/network check | 4 | PASS |
| 채팅 전송 | `PmChatPanel.vue`, `RoleChatPanel.vue` | `src/ipc/agent.ts` | `send_agent_message` | agent session 처리 | 4 | PASS |

평가: Vue에서 Rust까지의 호출 경로가 wrapper로 분리되어 있고, 직접 `invoke()` 남용 없이 도메인별 IPC 모듈을 경유한다. 현재 규모에서는 체인 깊이가 성능 병목으로 판단될 수준은 아니다.

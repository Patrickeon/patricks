# 01.분석/

> 산출물의 정의 , 하기에 설명한 산출물들은 폴더형태며 하위폴더로 관련 산출문서들을 저장한다.

| 하위 | 무엇 | 담당(R) |
|---|---|:---:|
| `AN-10_요구사항목록/` | 고객의 요구사항이 목록형태로 저장되어 진다. 특별한 양식은 없으며 고객의 워딩을 그대로 표현한다. 고객의 언어로 작성되어진다. | PM |
| `AN-20_요구사항정의서/` | 고객의 요구사항을 정의한다. | PM |
| `AN-40_요구사항명세서/` | 요구사항정의서의 요구사항 각건들에 대한 구현방안을 명시한다. | PM |
| `AN-50_요구사항추적표/` | 요구사항명세서에 적혀있는 각 건들에 대해서 요구사항을 구현한 산출물 또는 충족시킨 산출물의 물리적 위치를 링크한다. | PM |

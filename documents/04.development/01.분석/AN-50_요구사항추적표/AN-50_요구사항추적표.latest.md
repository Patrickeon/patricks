# AN-50 요구사항 추적표 — AgiTeamBuilder Desktop

> **목적**: 요구사항(AN-40) ↔ 설계(DS-10/20/40/50/60) ↔ 시험(TS-05) 간 정합 매핑을 단일 INDEX로 제공한다.  
> **입력**: AN-40 요구사항명세서, DS-10~DS-60 설계 산출물  
> **갱신 시점**: 설계 변경 시, 구현 완료 후, 시험 결과 확정 시

## 1. 매핑 구조

```
AN-40 요구사항 ID
  └→ 설계 산출물 (DS-XX §섹션)
       └→ 구현 산출물 (DV-XX)
            └→ 시험 항목 (TS-XX 시나리오)
```

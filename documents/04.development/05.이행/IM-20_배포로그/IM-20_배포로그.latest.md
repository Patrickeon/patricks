## 6. Windows 빌드 로그

Windows 빌드는 실행하지 않았다.

| 항목 | 값 |
|---|---|
| 실행 명령 | `cargo tauri build --target x86_64-pc-windows-msvc` |
| 실행 여부 | 미실행 |
| 사유 | 현재 환경이 macOS이며 Windows MSVC toolchain, Windows SDK, WebView2 Runtime 미구비 |
| 추가 제약 | 공통 Rust/Cargo toolchain도 미설치 |

## 7. 생성 파일 목록

빌드 실패로 신규 배포 산출물은 생성되지 않았다.

확인 명령:

```bash
find /Users/patrickukim/Projects/patricks/system/tos/src-tauri/target \
  -maxdepth 7 -type f \( -name '*.dmg' -o -name '*.app' -o -name '*.exe' -o -name '*.msi' -o -name '*.app.tar.gz' \) -print
```

확인 결과:

```text
생성 파일 없음
```

## 8. 결과 요약

| 구분 | 명령 | 결과 | 원인 |
|---|---|---|---|
| 도구 확인 | `cargo --version` | 실패 | cargo 미설치 |
| 도구 확인 | `rustc --version` | 실패 | rustc 미설치 |
| macOS 빌드 | `pnpm tauri build` | 실패 | `cargo metadata` 실행 불가 |
| Windows 빌드 | `cargo tauri build --target x86_64-pc-windows-msvc` | 미실행 | Windows 빌드 환경 미구비 |

## 9. 후속 재실행 명령

Rust toolchain 설치 후 다음 순서로 재실행한다.

```bash
source "$HOME/.cargo/env"
cargo --version
rustc --version
cd /Users/patrickukim/Projects/patricks/system/tos
pnpm install
pnpm tauri build
```

Windows 인스톨러는 Windows 빌드 머신에서 다음 명령으로 재검증한다.

```powershell
cargo tauri build --target x86_64-pc-windows-msvc
```

## 9. 후속 재실행 명령

Rust toolchain 설치 후 다음 순서로 재실행한다.

```bash
source "$HOME/.cargo/env"
cargo --version
rustc --version
cd /Users/patrickukim/Projects/patricks/system/tos
pnpm install
pnpm tauri build
```

Windows 인스톨러는 로컬 대신 CI로 검증 완료 (§6.2). 향후 재검증이 필요하면 다음 방식 중 하나를 사용한다.

```bash
# 방법 1: 태그 재푸시 (버전업 시)
git tag lugh-vX.Y.Z && git push origin lugh-vX.Y.Z

# 방법 2: 수동 트리거 (workflow_dispatch)
gh workflow run "Build Windows" --repo Patrickeon/patricks
```

**남은 작업**: 실제 Windows PC에서 NSIS/MSI 인스톨러 설치·앱 구동 육안 확인 (CI는 빌드 성공까지만 검증, 설치·실행 UX는 미검증).

## 3. cmux → Tauri IPC 전환 대조표

> 현행 bash+cmux 명령과 GUI 앱 대응 기능 매핑 — baseline 정합성 검증 기준.

| cmux 명령 | 대응 GUI 기능 | 설계 근거 |
|-----------|-------------|---------|
| `cmux send --surface N "..."` | RoleChatPanel.vue 메시지 입력+전송 | DS-50 §7, DS-60 §IPC |
| `cmux send-key --surface N Enter` | 자동 Enter 제출 (Rust AgentSession) | DS-20 §AgentSession |
| `cmux read-screen --surface N --lines 30` | RoleChatPanel.vue 출력 자동 표시 | DS-50 §7.2, DS-60 §이벤트 |
| `cmux browser --surface N <cmd>` | BrowserPanel.vue 내장 브라우저 | DS-50 §11 |
| `agiteam.sh` 부팅 전체 | Screen-03 BootView.vue 7단계 부팅 | DS-50 §4 |
| `security find-generic-password` (keychain) | OS Credential Vault (Keychain/CredMgr) | DS-20 §CredentialStore |
| `agiteam.json` 수동 편집 | Screen-02 SettingsView.vue GUI 편집 | DS-50 §3 |
| `project_state.yaml` 수동 편집 | Screen-07 설정 패널 YAML 편집기 | DS-50 §10 |
| bash 스크립트로 레드마인 curl | Screen-06 RedminePanel.vue CRUD | DS-50 §9, DS-40 §레드마인 |
| `.latest.md` 파일 직접 vim 편집 | Screen-05 DeliverableViewer.vue | DS-50 §8 |
| Python `find` 경로 탐색 | Rust `std::fs::read_dir` | DS-20 §FileDocument |

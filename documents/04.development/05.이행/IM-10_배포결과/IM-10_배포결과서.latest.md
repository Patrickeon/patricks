### 5.5 참고 사항 — Node.js 20 deprecation 경고

CI 실행 로그에 아래 경고가 있었으나 **빌드 결과에는 영향 없음** (Node24로 강제 승격되어 정상 실행됨):

> Node.js 20 is deprecated. The following actions target Node.js 20 but are being forced to run on Node.js 24: `actions/checkout@v4`, `actions/setup-node@v4`, `actions/upload-artifact@v4`, `pnpm/action-setup@v4`.

후속 조치는 9절 참조 (개선 항목으로 기록, 이번 릴리스 차단 사유 아님).

## 9. 후속 조치

1. **TS25-001·TS25-002 MG-40 전건 닫힘**: 완료 (2026-06-25)
2. **Windows 빌드**: ✅ CI 성공 완료 (2026-07-12, run `29189569590`). 남은 것은 실기(실제 Windows PC) 설치·구동 육안 확인.
3. **cargo PATH 가이드**: IM-20 배포 로그에 PATH 명시 추가 권고
   - `export PATH="/Users/patrickukim/Library/Caches/puccinialin/rustup/toolchains/stable-aarch64-apple-darwin/bin:$PATH"`
   - `.zprofile` 또는 프로젝트 공통 스크립트에 영구 등록 권장
4. **[개선 항목] GitHub Actions Node.js 20 deprecation 대응**: `actions/checkout@v4`·`actions/setup-node@v4`·`actions/upload-artifact@v4`·`pnpm/action-setup@v4`가 Node20 target이나 Node24로 강제 실행 중(현재 빌드 무영향). 각 액션 차기 메이저 버전 출시 시 업그레이드 권고. 상세는 IM-20 §6.3 참조.
5. **keyring OS 백엔드 feature 미지정 건**: `src-tauri/Cargo.toml`의 `keyring = "3"`에 플랫폼 feature(windows-native/apple-native 등) 미지정 — 현재 컴파일은 되나 mock(더미) 백엔드로 동작하여 실제 자격증명이 OS 보안 저장소에 저장되지 않을 가능성 있음. BE 확인 및 결함 등록 여부 PM 판단 대기 중 (DevOps 소관 아님, C=자문).

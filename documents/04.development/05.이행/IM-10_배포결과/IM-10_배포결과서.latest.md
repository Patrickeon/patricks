본 문서는 TS-60 릴리스 승인 이후 AgiTeamBuilder Desktop의 macOS DMG 인스톨러 생성 및 앱 실행 검증 결과를 기록한다.

### 3.1 cargo PATH 설정

```bash
export PATH="/Users/patrickukim/Library/Caches/puccinialin/rustup/toolchains/stable-aarch64-apple-darwin/bin:$PATH"
cargo --version  # cargo 1.94.1 (또는 유사 버전)
```

> **원인**: `~/.cargo/bin` 미존재. rustup이 `/Users/patrickukim/Library/Caches/puccinialin/rustup/` 경로에 별도 설치됨. DevOps 환경에서 PATH 미설정으로 v0.1 빌드 실패했으나, 해당 경로 사용으로 성공.

# documents/

> AgiTeam이 생성하는 프로젝트의 모든 산출물을 **단계·모드별로 분류해 저장**하는 최상위 폴더.
> 전체 폴더 트리의 권위 정의(SSOT)는 `00.standard/폴더구조.latest.md` 에 있다. 본 README는 **바로 아래 한 겹**만 안내한다.

| 하위 | 무엇 | 적용 모드 |
|---|---|---|
| `00.standard/` | 모든 모드 공통 기준 문서(RACI매트릭스·폴더구조 표준). 수정 없이 상속 | 공통 |
| `01.proposal/` | 제안 진입자료(PD) | Forward 전용 |
| `02.reverse/` | 인수 진입자료(RD) | Reverse 전용 |
| `03.management/` | 관리 산출물(착수준비 IN + 상시관리 MG) | 모든 모드 |
| `04.development/` | 개발 산출물(분석 AN → 설계 DS → 구현 DV → 테스트 TS → 이행 IM) | 개발 |
| `05.operation/` | 운영 산출물(OP) | 운영 모드 |

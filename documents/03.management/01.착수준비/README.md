# 01.착수준비/

> 착수 시점에 **일회성으로 작성**하는 관리 산출물(IN)을 둔다.

| 하위 | 무엇 | 담당(R) |
|---|---|:---:|
| `IN-10_수행계획서/` | 수행 체계·범위·일정·관리 계획 기준 문서 | PM |
| `IN-20_WBS/` | 산출물·활동을 분해해 일정·담당을 배분한 작업분해구조 | PM |
| `IN-30_착수보고서/` | 공식 착수를 보고·승인받는 문서 | PM |
| `IN-40_품질보증계획서/` | 품질 보증 활동 계획 | PM |
| `IN-50_위험관리계획서/` | 위험 식별·대응 계획 | PM |
| `IN-60_형상관리계획서/` | 산출물·코드 형상관리 계획 | PM |

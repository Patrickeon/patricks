# 02.상시관리/

> 프로젝트 내내 **상시 갱신**하는 보고서·관리대장(MG)을 둔다. 운영 모드에서도 필수로 누적된다.

| 하위 | 무엇 | 담당(R) |
|---|---|:---:|
| `MG-10_중간보고서/` | 진척 중간 보고 | PM |
| `MG-20_완료보고서/` | 프로젝트 완료 보고 | PM |
| `MG-30_주간보고서/` | 주간 활동 진척 집계 | PM |
| `MG-40_품질관리대장/` | 결함·품질 이슈 상시 기록 | QA |
| `MG-50_보안관리대장/` | 보안 점검·조치 상시 기록 | DevOps |
| `MG-60_변경관리대장/` | 변경 요청·반영 상시 기록 | PM |
| `MG-70_이슈및위험관리대장/` | 이슈·위험 상시 기록 | PM |
| `MG-80_회의록/` | 회의 결과 기록 | PM |
| `MG-90_ActionItem관리대장/` | 액션 아이템 추적 | PM |
